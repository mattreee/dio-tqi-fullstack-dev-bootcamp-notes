Casting

é a transformação de uma determinada variável de tipo menos específico para um tipo mais específico ou vice-versa;

Tipos
   > Upcast (implícito)
   > Downcast (explícito)

podem ser feitos em tipos primitivos e compostos; 

Ex.: 
   > long a; int i = 10; a = i;           (implícito)
   > int i; long a = 100; i = (int) a;    (explícito)
